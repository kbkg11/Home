
* 자바 1일차 공부

*** 기본 자료형 
자바언어에 이미 존재하고 있다. 기본자료형은 바로바로 메모리에 들어간다 <br>
(int, double, boolean, char)

*** 객체 자료형
여러가지 데이타들이 모여있는 복잡한 데이터이다. 객체자료형은 객체자료형이 실제로 들어있는 주소값이 들어간다.<br>
(string, System, ArrayList) 

*** 산술연산자
덧셈 : + <br>
뺄셈 : - <br>
곱셉 : * <br>
나눗셈 : / <br>
나머지 % <br>

*** 자동증감 연산자
증가 : i++
감소 : i--

*** 논리 연산자
AND연산자(둘다 참일때 참) : &&
OR연산자(둘중 하나만 참이여도 참) ||

*** 조건문
```
if(조건){
 참일 때 수행할 내용
} else if {
거짓일 때 수행할 내용
}
```
*** 반복문 
```
for(초기값, 조건식, 증감식){
반복 수행할 내용
}
```
